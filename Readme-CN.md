# Api-X

[English](Readme.md)

JDK 21

- `gradle run` 调试运行
- `gradle createDistributable` 打包

## 功能列表

- 导入导出 Postman Collection 格式数据
- 主题切换
- 请求运行日志
- Ctrl+K 全局搜索
- Ctrl+Tab 切换最近 Request
- 多 Env 切换和变量解析
- 文件夹 Auth 继承
